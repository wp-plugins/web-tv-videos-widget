A widget to display titles and links to the latest videos from TV channels and newssites (de/en) about a self-chosen topic in the sidebar.

== Description ==

This widget displays a list with links to the latest videos from TV channels and newssites about a certain topic in the sidebar. The topic can be freely chosen and can be a name of a popular person ("Britney Spears", "Steve Jobs", "Barack Obama", ....) you write about in your blog, a political or technical topic ("Egypt", "iPhone", ....), a name of a TV show ("60 minutes", "Tagesschau", ...) or anything you can imagine. Since this plugin uses the <a href="http://www.tvbookmark.de/articles/?page_id=252" target="_blank">Web-TV Video API</a> of the WebTV search engine TVBookmark, please test your search there (US: <a href="http://www.tvbookmark.info" target="_blank">tvbookmark.info</a>, DE: <a href="http://www.tvbookmark.de" target="_blank">tvbookmark.de</a>) to see if the results of your chosen topic are useful. Videos will be updated every 30 minutes. At the moment, the plugin support video results from the U.S. and Germany, displaying entries from Channels like ABC, CBS, FOX, CNN or MSNBC when english videos are chosen and entries from Channels like Das Erste, ZDF, RTL, Sat1, Pro7, Dritte Programme, Spiegel Online or Sueddeutsche.de when german videos are selected.

== Installation ==

ATTENTION! This plugin uses the WP_Widget class of WordPress introduced in v. 2.8, so it will not work in earlier WordPress versions.

1. Unzip downloaded `web-tv-videos-widget.zip` file
2. Upload resulting `web-tv-videos-widget` folder to the `/wp-content/plugins' directory
3. Activate the plugin through the `Plugins` menu in WordPress
4. Add the plugin (widget) to the chosen widget area and configure your plugin in the `Appearance -> Widgets` menu
